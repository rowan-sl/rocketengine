# rocketengine

Basic (and very unfinished) 3DOF (soon to be 6) rocket flight simulator

## Features

- graphing of rocket flight paths and charecteristics
- custom rocket engines (easily add new ones!)

## Dependancies

- build system
  - python
    - pathlib
    - xml
    - yaml
  - rust
    - see `Cargo.toml`
- simulator
  - see `Cargo.toml`
